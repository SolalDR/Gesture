# Gesture

## Install

### Clone the repository

```shell
git clone git@github.com:SolalDR/Gesture.git
cd Gesture
```

### Check the ruby version (2.5.0)

```shell
ruby -v
```

It must output something like "2.5.0". If not, install the right ruby version:

```shell
rbenv install 2.5.0
```

### Install dependencies

```shell
npm install && bundle
```

## Serve

```shell
middleman
```

## Build

```shell
middleman build
```
